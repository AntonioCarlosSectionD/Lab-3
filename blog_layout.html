discussion questions

1. How does creating a cohesive theme enhance the user experience of a website?

Creating a cohesive theme can make your website more visually appealing to look at. If the website did not have a cohesive theme, it can disrupt the user's experience since pages will clash together. Having a cohesive theme ensures that there will be a postive impact on the user so that they may feel compelled to visit again.


2. What are the key considerations when designing forms for user input?

Key considerations to include for forms of user input is to make sure these places stand out with prominent colors. That way, the user would know which space is meant for them to input. Additionally, it is also important to have text that says what the user should do so they do not put the wrong information.

3. How can multimedia content improve the engagement and information value of a blog post?

Multimedia content can improve engagement and information in a blog post because they are useful for conveying information in a quick manner. Users may lose attention and interest if all the blog has to offer is a wall of text. Multimedia such as images and videos can quickly give the user information while also making the blog more engaging.
